PIR Motion Based Lighting Control System

  • Sources & documentation

Built a system that's responsible for turning On/Off lights automatically based on motion detection using PIR Sensor.

When some one enters in the range of PIR sensor it triggers the signal and appropriate actions will be taken.

  • Technologies used:Arduino, Proteus, PCB Designing, Circuit Testing, sensor, Relays
  • PIR sensor outputs a certain voltage when it detect motion. This voltage is in the order of milliVolts
  • This output voltage will be amplified using OptoCoupler circuit(manuallly designed) to make sure it is enough to be read by the micro controller(typically 3.3v)
  • Once the Micro controller read the control signal an appropriate action will be taken(Relay On/Off)
