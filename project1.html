Digital Resume

  • Source Code

Digital version of my resume built and deployed using HTML, CSS and little bit of JavaScript.

It Contains all of Work History, Projects that I made, my Education details, Social Profiles etc.,

  • Technologies used:HTML, CSS, JavaScript and Github
  • Built with basic CSS Flex layouts, Custom CSS Properties, Dark & Light theme toggling using JavaScript
  • Theme settings will be stored in local storage. So, even user reloads/re-opens the last stored theme will be apllied
  • Deployments will be triggered automatically as soon as the new source code reaches the mainline branch
  • Deployed for free using: Github Pages
