Single Phase Multi-Level Inverter

  • Source Code

Designed a basic Single Phase Multi-Level Inverter to converts fixed DC supply to variable frequency AC supply.

The speed of AC machines responds to variations in input frequency.

  • Technologies used: Arduino IDE, Arduino Programming, PCB Designing, Circuit Functionality Testing, Proteus
  • First AC is converted into DC and then converted into variable frequency AC
  • Variable frequency is done using PWM technique
  • And for the pure sine output SPWM or SVM techniques are the better options
