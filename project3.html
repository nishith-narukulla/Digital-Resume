Energy Monitoring System using Raspberry pi

  • Any Other Links

Built a system that measures energy consumption and other electrical parameters like voltage, current and power factor using Raspberry pi micro-controller and sensors.

And Built a GUI using Python pyQT5 module to show the data to the consumer.

  • Technologies used:Python, QTdesigner, Proteus
  • Raspberry Pi completely programmable with Python and support wide range of Interfacing options
  • This system will be capable of monitoring four appliances and store data separately
  • Accuracy is much better
  • ADC's are used to convert sensor analog data to Digital
