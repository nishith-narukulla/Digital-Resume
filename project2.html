IOT web-based Temperature Monitoring System

  • Source Code

Built a system that senses temperature and sends data to web server using ESP-32 Micro controller.

This data will be plotted on the UI. And it is used to monitor the room temperature remotely from any in the world and take appropriate actions like turning on cooling systems, turnning off heaters or adjusting temperature off the other systems.

  • Technologies used:Arduino, Proteus, HTML, CSS, Google Firebase and Sensors
  • ESP-32 acts as medium between temperature sensor and the web server
  • Circuit Board design is done using Proteus Software
